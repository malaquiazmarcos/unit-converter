Unit Bridge API

With this API you can convert all units in your mind.

Current Version: v1

ABOUT THIS PROJECT

This project was designed to simplify unit conversion as much as possible, keeping in mind developers just starting their training or students.

IMPORTANT INFORMATION

  • Before converting units, verify that the format is correct at the endpoints (e.g., "cm", not "centimeter").
  • In endpoints that accept data, if a parameter like {value} is only descriptive, you must replace it with the actual value without the curly braces.
  • Rate Limit: Each IP address is limited to 10 requests per minute.
  • Some units that use μ or Ω may be seen as u and O respectively.
  • Note: There may be grammatical errors in English.

Units Converter

Convert units based on user input.

Endpoint (v1):

http://127.0.0.1:8000/v1/unit-convert?value={value}&unit_type={unit_type}&unit_from={unit_from}&unit_to={unit-to}

HTTP Method:

GET

Query Parameters:

  • value (float, int): Numeric value to convert. Required.
  • unit_type (str): Magnitude input for data search (e.g., 'temperature', 'length'). Required.
  • unit_from (str): Abbreviation of input unit (e.g., 'C', 'mm'). Required.
  • unit_to (str): Abbreviation of desired unit (e.g., 'K', 'GB'). Required.

Responses:

  • 200 OK: Returns the converted value. 
    {
    "value": 1.0,
    "unit_type": "temperature",
    "unit_from": "C",
    "unit_to": "K",
    "result": 274.15
}
  • 400 Bad Request: Invalid or empty parameters. 
    {"error": "Value is not a valid number"}
    {"error": "Requested data missing"}
  • 404 Not Found: Units data not available. 
    {"error": "Requested data not found"}
  • 429 Too Many Requests: Rate limit exceeded. 
    {"error": "Limit reached. Maximum 10 requests per minute."}
  • 500 Internal Server Error: An unexpected server error occurred. 
    {"error": "Internal server error"}

        ...
    },
    ...
}
  • 404 Not Found: Units data not available.

  • 500 Internal Server Error: An unexpected server error occurred. 
    {"error": "Internal server error"}

Description of One Unit

Returns a short description of one unit requested by the user.

Endpoint:

http://127.0.0.1:8000/v1/short-description?unit_type={unit_type}&unit={unit}

HTTP Method:

GET

Query Parameters:

  • unit_type (str): Magnitude input for data search (e.g., 'length', 'data'). Required.
  • unit (str): Unit to show description (e.g., 'cm', 'MB'). Required.

Responses:

  • 200 OK: Returns a successful description. 
    {
    "MB": "Common file size unit (1 MB = 1,000,000 bytes). Used for storage devices."
}
  • 404 Not Found: Unit description not found. 
    {"error": "Requested data not found"}
  • 500 Internal Server Error: An unexpected server error occurred. 
    {"error": "Internal server error"}
